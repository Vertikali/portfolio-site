Портфолио верстальщика

С сентября 1998 по июнь 2000 года, учась в Костанайском ПТШ-3 на электромонтажника, освоил азы программирования на языке Бейсик (версия ВИЛЬНЮС, 1986. 07.24.). На микроЭВМ БК-0011М.

Весной 2003 года, в агенстве "CENTRE", обучился специальности Оператора ПК (Word, Exel, Интернет, электронная почта).

С октября 2014 по апрель 2017 самостоятельно учился по книге Р.Лафоре."Объектно-ориентированное программирование на С++"(2004). Проверяя написанные мной программы в CodeBlocks изучил базовые конструкции языка.

В сентябре 2017 по книге В.Дронова "HTML5, CSS3 и Web2.0"(2011), используя "Блокнот" и Google Chrome, сверстал первые html страницы.

Летом 2018 практикуясь в вёрстке, по видео-урокам "HTML & CSS Александр Пауков" на YouTube, приобрёл опыт создания сайтов с PSD макетов. Ипользовал Notepad++ и FireFox.

С августа 2018 по ноябрь 2018 года, повторяя примеры из книги К.Аквино, Т.Ганди."Front-end. Клиентская разработка"(2017), познакомился c Visual Studio Code и Node.js.

В 2019 году, с марта по сентябрь, по макетам формата PSD, написал четыре лендинг страницы для своего портфолио.

В июле 2019 по видео-урокам "Git+GitHub ...", канала ITDoctor в YouTube, изучил основные инструменты GitHub-а.
